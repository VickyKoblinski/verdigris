## Usage
Collection of SVG icon files converted to `SvgIcon` components. See the [SvgIcon](/packages/icons/components/SvgIcon) section to make your own SVG icons.

## Examples
Each icon can be imported as a property of this package.

```js
import { CloseIcon } from '@andrew-codes/verdigris-icons';
```
