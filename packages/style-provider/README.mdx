# StyleProvider
The `StyleProvider` component is intended to be used only once, at the top of your application's component tree. It effectively is what provides a `react-fela` renderer to all components within the tree via React's context API.

> Note that you **must** use the `StyleProvider` in order to leverage other components in Verdigris. You only need a **single** `StyleProvider` node in your component tree.

## Example
```jsx
import React from 'react';
import ReactDOM from 'react-dom';
import StyleProvider from '@andrew-codes/verdigris-style-provder';
import App from './components/myApp';

ReactDOM.render((
  <StyleProvider>
    <App />
  </StyleProvider>
), document.getElementById('#app'));
```
