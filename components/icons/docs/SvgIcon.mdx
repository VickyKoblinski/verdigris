## Usage
Provides the ability to create an icon component from SVG markup. To enable importing SVG files directly as a custom SvgIcon, there is a webpack loader, `@andrew-codes/verdigris-svg-icon-loader`, that will accomplish this.

## Examples
Use pre-existing SVG markup with the SvgIcon component. Simply provide the SVG markup as the child.

```js
import SvgIcon from '@andrew-codes/verdigris-icons';
```
